# 601939.SS 深度研究报告：防御之锚，估值修复可期

**报告日期：2026年7月19日**
**评级：推荐（维持）**
**目标价区间：9.59元 - 12.33元（中枢10.96元）**
**当前股价：10.16元**

## 投资要点

**核心投资逻辑**：作为系统重要性银行，建设银行凭借44.65%的超高净利润率和稳健的资产质量，在宏观承压背景下成为机构资金的“防御性避风港”。当前0.76倍PB、7.78倍PE已反映盈利增速放缓预期，但未充分体现其逆周期调节下的政策红利与“破净”后估值修复的期权价值。我们认为，在避险情绪持续与机构共识托底下，股价具备温和上涨空间。

- **目标价区间**：9.59元 - 12.33元（中枢10.96元），较当前股价存在约7.9%的潜在上涨空间。
- **评级与催化剂**：推荐。核心催化剂包括：（1）市场系统性风险下的持续避险资金流入；（2）分红季在即，高股息吸引力凸显；（3）机构一致看好形成的底部支撑。
- **主要风险**：（1）宏观经济恶化超预期，冲击资产质量；（2）货币政策转向导致利率环境不利；（3）大盘反弹后防御资金流出引发“补跌”。

---

## 一、公司概况与业务分析

建设银行（601939.SS）是一家市值超9747亿元的超大型上市商业银行，2025年净利润高达3389.1亿元，净利润率达44.65%。尽管业务简介数据暂不可用，但基于其万亿级资产规模和极高负债率（92%），可判断其业务核心为存贷款净息差收入及中间业务收入。

- **行业地位**：作为系统重要性银行，其在货币信贷、跨境支付等领域具有准国家信用背书。资产负债规模与盈利能力在A股市场中居于头部。
- **竞争壁垒**：主要包括（1）低成本负债优势（庞大网点与零售存款基础）；（2）国有大行在不良资产处置和政府项目融资方面的资源禀赋；（3）机构高度共识的“防御属性”品牌溢价。
- **近期重要变化**：2026年7月17日A股市场出现极端恐慌（双创指数跌超7%，超5000只个股下跌），银行板块逆势走强，建行股价表现相对强势，强化了其作为“避风港”的市场叙事。

## 二、财务分析与盈利质量

### 2.1 收入与利润

2025年全年，公司实现营业收入7610.5亿元（同比+1.88%），净利润3389.1亿元（同比+0.99%）。2026年一季度营收增速反弹至11.15%（受低基数及信贷开门红驱动），净利率为41.1%。整体呈现“稳健、低增、韧性”特征。

### 2.2 盈利能力与ROE拆解

- **净利润率**：2025年达44.65%，且近5年呈持续上升趋势（从2021年36.87%提升至44.65%），表明成本管控与定价能力不断优化。这一数据在银行业中属于极高水平，反映其强大的盈利“护城河”。
- **ROE**：2025年全年ROE为10.04%，2026年Q1年化后约9.8%，与行业平均ROE基本匹配。ROE处于10%中枢，是支撑当前PE估值的核心锚点。

### 2.3 现金流质量

2025年经营活动现金净流量高达21012.4亿元，较2024年的3380.2亿元巨幅增长。这种剧烈波动是银行经营现金流的典型特征（受资产负债期限错配与市场间资金拆借影响），不代表经营恶化。但需关注现金流巨量波动的具体构成，若主要由短期套利驱动，则对可持续性构成隐忧。**自由现金流（FCF）数据暂不可用，无法做DCF估值**。

### 2.4 资产负债表健康度

截至最新数据，资产负债率约92%（基于负债/资产计算），略高于行业中枢，属银行正常经营模式。由于缺乏不良贷款率、拨备覆盖率等明细数据，无法评估具体资产质量，但财务健康度评分（8/10）和系统重要性身份暗示其风险管控能力较强。

### 财务分析师核心结论

- **正向**：净利润率持续上行至44.65%，盈利能力极强且稳定。
- **负向**：营收与净利润增速处于低位（1%-2%），增长弹性不足。
- **风险**：现金流波动巨大，需关注具体构成；宏观风险对资产质量潜在冲击。

## 三、市场情绪与预期

### 3.1 当前市场共识

主流叙事是“防御性资产与高股息避风港”。在7月17日A股恐慌下跌中，资金从高波动成长股流入银行板块，市场定价逻辑为**系统性风险下的避险交易**。11家机构全部给出“买入/增持”评级，目标价上限12.49元，形成强底部共识。

### 3.2 预期差分析

- **温和低估**：市场定价（PE 7.78倍）充分反映了盈利增速放缓（2-3%）的悲观预期，但未充分定价其无形资产价值，包括品牌、牌照和低成本负债优势。情绪分析师认为，当前估值可能低估了其在逆周期政策下的“政策红利”期权价值。
- **过度定价的风险**：市场过度地将所有银行股“一视同仁”视为宏观风险载体，而忽略了建设银行作为系统重要性银行，在不良资产处置上的国家信用背书和资源禀赋。
- **被忽视的风险**：货币政策转弯风险。若未来通胀或汇率压力导致利率上行，虽利好净息差，但可能引发资产价格重估，对持有大量债券和长期资产的银行构成冲击。

### 3.3 情绪面支撑与压制

| 支撑因素 | 压制因素 |
| :--- | :--- |
| 防御属性：系统性风险下的资金避风港 | 宏观系统性风险担忧 |
| 机构共识强，11家全部买入/增持 | 盈利增速低（<3%），上涨弹性不足 |
| 低估值+高分红预期吸引稳定资金 | 大盘反弹时可能资金外流导致“补跌” |
| 当前情绪偏正面（情绪评分+35） | PE处于近10年历史最高分位（97.3%） |

## 四、估值分析与目标价

### 4.1 估值方法及核心假设

由于银行股自由现金流波动剧烈且不可靠，DCF模型不适用；缺少可比公司数据，无法进行横向对比。**本次估值采用PE估值法（核心）**，辅以PB（0.76倍，当前破净状态）作为底线参考。

**核心假设**：
- **合理PE中枢**：基于2025年ROE 10.04%和预测EPS年均增速~3%，结合银行股历史区间，取8.0倍。考虑国有大行地位和防御属性溢价0.5倍，最终**合理PE为8.5倍**。
- **PE区间**：基于情绪支撑取上限9.0倍（与机构隐含目标价PE基本一致），基于盈利下行风险取下限7.0倍（保守情景）。
- **预测EPS**：直接采用11家机构一致预期：2025E（实际）1.30元，2026E 1.32元，2027E 1.37元。

### 4.2 目标价计算

以2027年预测EPS（1.37元）为估值基准（覆盖两个完整财年，体现中长期价值）：

- **区间下限（保守）**：1.37元 × 7.0倍 = **9.59元**
- **区间上限（乐观）**：1.37元 × 9.0倍 = **12.33元**
- **中枢目标价**：（9.59 + 12.33）/ 2 = **10.96元**

**当前股价（10.16元）较中枢目标价有约7.9%上涨空间**。目标价上限（12.33元）与机构上限（12.49元）高度吻合，下限（9.59元）略低于机构下限（11.0元），体现了模型对估值下行风险的保守判断。

### 4.3 估值结论

| 项目 | 数值 |
| :--- | :--- |
| 目标价格区间 | 9.59元 ~ 12.33元 |
| 中枢目标价 | 10.96元 |
| 当前股价 | 10.16元 |
| 上涨/下跌空间 | +7.87%（相对中枢） |
| 估值评级 | **推荐**（假设强度：中） |

## 五、投资结论

### 5.1 综合评级：推荐

**核心逻辑**：建设银行具备“高股息+低估值+系统重要性”三重安全垫。当前市场避险情绪为股价提供了短期支撑，而长期看，其44.65%的超高净利润率和机构共识提供的底部托底，使得现价具有防御性买入价值。当前股价低于中枢目标价，存在温和的估值修复空间。

### 5.2 核心推荐/回避逻辑

- **推荐买入场景**：
  - 追求绝对收益、对回撤容忍度低的投资组合，可作为底仓配置。
  - 看好市场系统性风险延续，需要避险工具。
  - 寻找低估值、高股息的现金等价物替代。

- **建议回避场景**：
  - 预期市场风险偏好快速回暖，高贝塔资产将获得超额收益（银行股可能跑输且“补跌”）。
  - 对银行资产质量高度悲观，认为宏观经济恶化将导致不良率飙升。
  - 追求高弹性成长，对股息和防御属性不敏感。

### 5.3 关键催化剂与时间窗口

1. **短期（1-3个月）**：分红季临近，高股息吸引力释放，可能触发资金增配。
2. **中期（3-6个月）**：若经济数据持续疲软，政策层面出台降准或定向宽松措施，作为信贷投放主力的国有大行将直接受益，估值修复逻辑可能强化。
3. **风险触发点**：若后续披露的不良贷款率或关注类贷款占比超预期上行，当前估值中枢（8.5倍PE）将面临下修压力。

### 5.4 风险提示（必须具体）

1. **宏观经济超预期下行风险**：若GDP增速持续走低、房地产行业风险蔓延，银行不良贷款率和拨备计提将显著上升，盈利预测（EPS 1.37元）面临下行风险，目标价可能跌至下限9.59元以下。
2. **估值体系重构风险**：当前PE处于近10年97.3%分位，属历史极高位置。若市场风格系统性转向成长股，银行股的历史估值中枢可能下移，PE可能压缩至7.0倍以下。
3. **情绪反转风险**：当市场恐慌情绪消退，资金从防御性银行股流出“抄底”超跌成长板块时，建行可能面临阶段性“补跌”，且跌幅往往在市场企稳初期最为显著。
4. **货币政策与利率风险**：若通胀压力或汇率波动导致货币政策收紧，利率短期上升虽利好息差，但可能引发债券市场重估，对银行持有的长期资产造成冲击。
